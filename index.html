欢迎来到 REST CookBook

REST 非常性感！ 并且 REST 最终被全世界 的 API 程序员重新发掘。然而 REST 并不全如其第一眼看起来那样的简单。 处理 HATEOAS（Hypermedia As The Engine Of Application State），按需编码并统一接口可以非常微妙。当一切开始变难的时候，很多人都会折衷为 「不全 RESTFul（not-so-RESTful）」的形式。它当然并不需要那样（变得 RESTful）。当你开始了解 REST，你将会喜欢它。

The RESTful Cookbook 网站受到 Dean Wilson 所启发，他创建诀窍手册来帮助人们解决使用 「Puppet 配置管理工具」所遇到各种问题。而这网站，将专注解决人们尝试创建 RESTful API 的觉问题。

这个网站由 Joshua Thijssen 创建并发布在 github，如果你愿意可以经常添加或者更新上面的内容。伴随更多的贡献， 这个网站将可能成为你以及大家寻找 RESTful 相关解决方案的的地方。
